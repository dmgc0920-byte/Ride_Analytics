Ride_Analytics

This repository contains the code for Ride_Analytics, a data analytics platform for ride-sharing services. The platform provides insights into ride patterns, driver performance, and customer behavior.

1. The project includes data preprocessing, exploratory data analysis, data visualization and probability modeling.

Versión en Español

Este repositorio contiene el código para Ride_Analytics, una plataforma de análisis de datos para servicios de transporte compartido. La plataforma proporciona información sobre patrones de viaje, rendimiento de los conductores y comportamiento de los clientes.

1. El proyecto incluye preprocesamiento de datos, análisis exploratorio de datos, visualización de datos y modelado de probabilidad.
